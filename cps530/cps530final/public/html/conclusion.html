Conclusion


Bulma and NodeJS make a great pair for front-end and back-end development. Working with both has been a great learning experience because they are easy to learn even if one has no prior experience with them. Additionally there were lots of resources online for both frameworks to help us along the way.

Our experience with Bulma

Our experience with Bulma


Using Bulma, creating good-looking dynamic webpages was a breeze.
Bulma is very easy to install and set up, and being flexbox-based means it is very customizable.

Our experience with NodeJS

Our experience with NodeJS


Working with NodeJS is very simple and even beginners can pick it up and get started within minutes. The framework is easy to install and comes with the built in Node package manager (NPM). NPM makes it very easy to search for and install external libraries. Additionally NPM comes with dedicated command line tooling to make NodeJs development easier.
We used the Express framework on NodeJs to create our backend server. It had great documentation and was very intuitive to use. Express is structured by default in the MVC pattern making it very convenient.

What is NodeJs?


Released in 2009 by Ryan Dalh, NodeJs is a JavaScript environment created to run JavaScript on the server. Prior to 2009 JavaScript was mainly used in the browser to improve the interactivity of static web documents. In addition to front-end JavaScript NodeJs makse use of an updated eventloop and a new asynchronous low level IO functions.

How Popular is NodeJs?

How Popular is NodeJs?


NodeJs is one of the most popular GitHub repositories ever with over 65000 stars. It currently ranks 29th in terms of total stars on github and is the most popular serversided JavaScript framework.

What are the strengths and weaknesses of NodeJs?

Strengths and Weaknesses


Strenghts

  • NodeJs and its associated package management tools simply developer works.
  • New ASYNC architecture allows easy IO management.
  • Easy transition for front-end developers.
  • High Performance for Apps that have high transaction volumes
  • The Support of Large and Active Development Community
  • Provides Advance Caching Mechanisms
  • Great Open Source Support
  • Handles the Requests Simultaneously
  • Node.js is Highly Extensible

Weaknesses

  • Application Programming Interface (API) is Not Stable.
  • Has many of the drawbacks of traditional JavaScript.
  • Does not have a Strong Library Support System for certain domains.
  • Asynchronous Programming Model is hard for beginners.

Where is NodeJs used?

Which developers mostly use NodeJs and for what kind of websites?


The framework is based heavily on frontend JavaScript it was initially popular with designers and frontend web developers. However as the project matured, it grew in popularity with traditional backend developers. As a general purpose server sided framework NodeJs is used on a variety of companies which include:

  • Netflix uses NodeJs to prototype and A/B test new builds of their site.
  • The core Linkedin app has a NodeJs backend.
  • The Walmart mobile app backend is also built ontop of NodeJs.

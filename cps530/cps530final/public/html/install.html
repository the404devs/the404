How to install Bulma using a Content Delivery Network.


  1. Follow this Link to find the official Bulma CDN.
  2. Click on /npm/bulma@0.8.0/css
  3. Select /npm/bulma@0.8.0/css/bulma.css
    This will select the most basic configuration of Bulma, which we recommend for beginners.
  4. Click on Show & Configure All Links
    Located to the right.
  5. Select HTML output on the new screen
  6. Copy and paste the generated link on to any HTML page in the head sections to begin using Bulma
    Note, no configurations of Bulma come with any JS.
    It's upto each developer to add his/her own JS.

How to install NodeJs.

How to install NodeJs.


Steps

  1. Download the windows installer from nodejs.org
    We recommend Version 12.13.1 or any other Long Term Support (LTS) version for the best compatibility.
  2. Run the installer
  3. Follow installer propmpts
    For beginners we recommend keeping the default installation settings
    Keep hitting next and accept the TOS and License agreements untill installation is complete.
  4. Test the installation
    Open your preferred terminal and type "node -v" to make sure the correct version was installed
    Congradulations! You have succesfully installed NodeJs and NPM on to your machine.

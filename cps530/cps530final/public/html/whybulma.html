How Popular is Bulma?


Bulma is the third most popular css framework on Github. There are roughly 37,000 developers star the project on Github

What are the strengths and weaknesses of Bulma?

Bulma's Strengths and Weaknesses


Bulma is based on Flexbox and much more customizable compared to other CSS frameworks. it also has a big developer community. on the downside Bulma is not supported in older browsers

  1. Since it is flexbox, it is widely supported across desktop and mobile browsers. Flexbox also makes it easy to create repeatable elements
  2. Bulma also have no default styling, unlike Bootstrap or Material which have rigid styling. It is also small and pure css, which you can modify by using Saas.
  3. Bulma have a big developer community that are ready to help you build your next website
  4. Weakness of bulma is, it only partially supports older browsers like Internet Explorer. this means that it will run slower on IE10.

Which developers mostly use Bulma and for what kind of websites?


Bulma is used primarily by front end developer. since Bulma is based on flexbox, it can be used for any kind of website ranging from content heavy website to static website. Because it is easily modifiable via Sass, the aesthetic of the website would look very creative.
